# Eye of Beholder - Dungeon Crawler Game

Un jeu de dungeon crawler en JavaScript inspiré d'Eye of Beholder.

## Fonctionnalités

- 6 emplacements pour personnages
- Déplacement dans une tour avec plusieurs étages
- Chaque étage contient 12 pièces avec un escalier
- Combat avec des monstres
- Équipement pour les personnages
- Gain d'expérience lors des combats

## Comment jouer

1. Ouvrez `index.html` dans votre navigateur.
2. Utilisez les boutons de contrôle pour vous déplacer dans les pièces.
3. Combattez automatiquement les monstres qui apparaissent.
4. Montez les escaliers pour accéder aux étages supérieurs.

## Structure du projet

- `index.html`: Page principale du jeu
- `css/style.css`: Styles CSS
- `js/character.js`: Gestion des personnages
- `js/dungeon.js`: Génération et gestion du donjon
- `js/combat.js`: Système de combat
- `js/game.js`: Logique principale du jeu

## Développement

Le jeu utilise du JavaScript vanilla avec Canvas pour le rendu. Ouvrez simplement `index.html` dans un navigateur pour jouer.
